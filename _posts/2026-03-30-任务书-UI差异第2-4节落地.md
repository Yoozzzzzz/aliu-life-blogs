## 1. IN（必须 / 应交付）

### 1.1 对应差异文档 **第 2 节 · 色彩系统**

| 编号 | 事项 | 交付要点 |
|------|------|----------|
| C2.1 | 封面主导动态取色（可选） | 设置中增加「**随封面着色**」开关（默认关亦可，默认开需评估对比度）。开启后根据当前曲封面（或主色 `coverColor` 兜底）生成 **CSS 变量**（如背景光晕、渐变），应用于正在播放相关区域；**须**在封面过亮/过暗时用启发式保证主文案对比度可读。 |
| C2.2 | 亮暗封面与底色的协调 | 在 C2.1 开启时，对提取亮度做简单分档（如明度阈值），**局部**加强偏深底或描边，避免白字糊在浅封面上或黑字不可读；与 `prefers-reduced-motion` 协调（可减少色彩过渡动画）。 |
| C2.3 | 日间天蓝（sky）轻装饰（可选增强） | 在 **不破坏** 现有令牌前提下，可为 `sky` 主题增加**极轻量**背景光晕/网格（性能友好、可关），对齐报告演示气质；**禁止**大色块遮挡内容。 |
| C2.4 | 年度音乐颜色 / 听行为叙事 | **剔除** —— 属后端与产品里程碑，**第 5 节范围**，本任务不做。 |

### 1.2 对应差异文档 **第 3 节 · 布局与信息架构**

| 编号 | 事项 | 交付要点 |
|------|------|----------|
| L3.1 | 冷启动「尽快可听」 | 在合规前提下缩短路径：例如首次进入应用时，若有**最近播放**且策略允许，**可选**直达 `/now` 或首页显著停留条一键续播；行为与 `docs/前端软件开发需求设计文档.md` **F1.2** 表述一致，并在设置或关于中可说明（若需「关闭直达播放页」可配置则更佳，非强制）。 |
| L3.2 | 列表与加载体验 | 主要列表页（首页/发现/黑胶/歌单等）在数据未到时展示 **骨架或轻量占位**，避免白屏；**不做**依赖推荐引擎的「智能排序」（属第 5 节 / 后端）。 |
| L3.3 | 播放页：歌词优先与评论折叠 | `NowPlaying` **默认**以歌词区为主视觉路径；评论/扩展信息 **默认折叠**，展开符合 **F4.2**；交互不增加广告或 Feed。 |
| L3.4 | 手势增强（在 Web 可行范围内） | 在 **正在播放** 或迷你播放器相关区域评估 **上下滑** 切上/下一曲（需防与竖向滚动冲突，可限定手势热区）；左右滑与现有返回手势协调；无法可靠实现时文档注明降级原因。 |
| L3.5 | 底部 Tab 数量重构（三 Tab） | **本任务默认 OUT**——与差异文档第 5、7 节一并 **不强制执行**；若产品后续强制三 Tab，**另起任务书**。本任务仅要求 **不破坏** 现有四 Tab 信息架构下的上述体验。 |
| L3.6 | 「情感插槽」 | **仅前端占位**：例如在播放页或歌单头图下预留 **一块可空的文案/卡片区域**（本地静态 copy 或空白），**不**接运营 API、**不**做乐评拉取（避免落入第 5 节）。 |

### 1.3 对应差异文档 **第 4 节 · 动效**

| 编号 | 事项 | 交付要点 |
|------|------|----------|
| M4.1 | 动效档位 | 设置中提供 **极简 / 标准（可选轻量黑胶旋转等）** 至少两档；**禁止**默认开启 Shader 级「光波」与高 GPU 粒子。 |
| M4.2 | 轻量黑胶 / 封面呼吸 | 在「标准」档用 **CSS** 实现可选的慢旋或慢缩放，遵守 `prefers-reduced-motion` 时关闭或降为静态。 |
| 加载与过渡 | 列表/页面关键加载使用 **呼吸式或骨架** 占位，与 `tokens.css` / `--duration-ui` 一致。 |
| M4.3 | 「性能优先」 | 设置中开关：开启后强制 **极简** 动效，并可在实现允许时用简单启发式（如 `hardwareConcurrency`、内存 hints）**默认**帮用户预选档（须可用手动覆盖）。 |
| M4.4 | AI、BPM、频谱驱动动效 | **剔除**——属第 5 节或远期单列，本任务不做。 |

## 3. 验收标准（AC）

| AC | 描述 |
|----|------|
| AC-1 | 关闭「随封面着色」时，视觉与现版主题令牌一致，无回归。 |
| AC-2 | 开启「随封面着色」时，主要标题/歌词在深浅封面上均可读（抽样 3 组深浅封面自测）。 |
| AC-3 | `prefers-reduced-motion: reduce` 下无强循环动画；动效档位切换即时生效。 |
| AC-4 | 主要列表在慢网下可见骨架/占位，无长时间空白无反馈。 |
| AC-5 | 播放页首次进入为歌词优先路径，评论默认折叠，符合 F4.2 口径。 |
| AC-6 | 本任务**不**增加对 Alger / `server/` 新业务接口的依赖；若有任选增强，须在 README 或开发说明标注。 |

## 5. 风险与依赖

| 风险 | 应对 |
|------|------|
| Web 上「上下滑切歌」与滚动冲突 | 热区收窄或仅全屏播放页启用；文档说明降级 |
