# 静渊项目 — 后端（音源 API）能力范围

本文档基于 [`docs/后端接口文档.MD`](./后端接口文档.MD)（网易云 Node API **Alger**  fork 说明）与 [`server/README.MD`](../server/README.MD)，描述**当前音源进程启动后，HTTP/模块层面大致能覆盖的能力域**，便于产品与前端对照规划；**不替代**官方接口逐条保证与长期可用性说明（以实际联调与网易侧变更为准）。

---

## 1. 定位与边界

| 项目 | 说明 |
|------|------|
| **实现形态** | `netease-cloud-music-api-alger` 提供的 **HTTP API**（默认端口可配，如 `PORT=4000`），亦可在 Node 内 **按模块名** 程序化调用（返回 `status` + `body`）。 |
| **在仓库中的角色** | `server/` 负责任务拉起与扩展；生产/浏览器场景通常需自研 **BFF/网关**（鉴权、限流、缓存、错误模型），见 `server/README.MD`。 |
| **能力来源** | 接口能力对齐（并扩展自）NeteaseCloudMusicApi 一类封装；文档内列出约 **300+ 项** 具体功能点，以下为 **按业务域归纳** 的能力范围。 |
| **不负责的内容** | 不等同于网易官方开放平台 SLA；不自动保证全接口永久可用；**法律与平台义务**由使用方自担（见 `.cursor/agent/AGENTS.md` §5.2、`server/README.MD`）。 |

---

## 2. 能力域总览（能做什么）

### 2.1 账号与登录

- 手机登录、刷新登录、验证码发送与校验、注册/修改密码、**二维码登录**、退出登录、**登录状态**查询。
- 绑定/更换手机、检测手机号是否注册、初始化昵称、重复昵称检测等账号侧辅助能力。

### 2.2 用户资料与会员形态

- 用户信息、等级、徽章、**用户状态**（查询/可设状态列表/同状态用户/编辑）、绑定信息、**VIP 信息**与成长值、任务与领取记录。
- 云贝（签到、任务、收支、账户、推歌等）、云豆、音乐人相关数据与任务、**账号云豆**等增值体系只读或部分交互类接口（以文档条目为准）。

### 2.3 搜索与发现入口

- **综合搜索**、**搜索建议**、默认搜索词、热搜（简略/详细）、**搜索歌手**、**播客/声音搜索**、**听歌识曲**。
- 首页「发现」、圆形图标入口等聚合型数据接口。

### 2.4 推荐与个性化播放

- **每日推荐歌单/歌曲**、**历史日推**（可用日期与详情）、**私人 FM**（含模式选择）、**心动模式/智能播放**。
- 各类推荐：歌单、新音乐、电台、节目、MV、视频、电台个性推荐等。

### 2.5 歌曲、歌词与播放资源

- 歌曲详情、**歌词**、**副歌时间**、相似歌曲、听了此歌的最近用户、**音乐是否可用**、**音质详情**、红心数量、**是否已喜欢**。
- **播放地址 / 客户端下载 URL**（含文档中的新版接口）、**云盘歌曲信息匹配纠正**、本地文件匹配网易歌曲信息等。

### 2.6 专辑、歌单与曲库组织

- 专辑内容与评论、专辑动态/销量/数字专辑详情、收藏/取消收藏专辑、已收藏专辑列表。
- 歌单详情、**获取歌单全部歌曲**、新建/删除/更新歌单（名、描述、标签、封面、顺序）、**收藏/取消收藏歌单**、歌单分类、精品歌单与标签、网友精选碟、**歌单导入**（元数据/文字/链接及任务状态）、相关歌单与推荐、歌单收藏者、**调整歌单与歌曲顺序**。

### 2.7 歌手、MV 与视频（网易体系内）

- 歌手详情/描述/单曲/MV/专辑/**全部歌曲**/热门 50 首、相似歌手、歌手粉丝与数量、**歌手动态**、艺人百科简要信息。
- MV 与视频：排行、数据、播放地址、详情、相关、评论、收藏、点赞转发评论数、网易出品 MV、全部 MV 等。

### 2.8 排行榜与榜单

- **排行榜**、**所有榜单**及**内容摘要**、新歌速递、歌手榜、各类电台/MV/视频榜单（文档中电台节目榜、24 小时榜等多条）。

### 2.9 评论与互动

- 歌曲/专辑/歌单/MV/电台节目/**视频**/动态/**楼层**等场景的 **评论列表、发送/删除、点赞、热门评论**、**抱一抱**及抱一抱列表等。

### 2.10 社交、动态与通知

- 用户动态（列表、转发、删除、分享到动态）、动态评论、热门话题、话题详情与热门动态、云村热评等。
- 关注/粉丝/互关判断、用户关注列表、粉丝列表、**最近联系人**。
- **私信**（含私信音乐、私信歌单）、**通知**（私信/评论/@/系统通知等聚合类接口）。

### 2.11 电台、播客与声音

- 电台推荐、分类、订阅、详情、节目与节目详情、banner、用户/热门电台、付费精品榜等多类电台榜单与排行。
- 播客：**声音列表/上传/排序/删除**、播客详情、**声音歌词**、相关播客能力；与电台节目形成文档中的长篇能力列表。

### 2.12 云盘与个人资源

- 云盘列表与详情、**上传**、**删除**、**导入歌曲**、云盘数据详情等。

### 2.13 视频生态与第三方 B 站（文档单列）

- 网易侧：**视频详情/播放/标签/分类/列表/推荐/历史播放**、视频评论与收藏、点赞视频列表、视频歌单增删等。
- **B 站**：搜索视频、视频详情、播放地址（文档「bilibili」三节）。

### 2.14 运营与杂项能力（节选）

- **Banner**、**签到**、**听歌打卡**、签到进度、**批量 batch 请求**。
- **曲风**（列表、偏好、详情及歌曲/专辑/歌单/歌手维度）、**乐谱**列表与内容、**音乐百科**简要信息（歌曲/专辑/歌手/MV）。
- **音乐日历**、**黑胶时光机**、**年度听歌报告**、**听歌足迹**（今日/总时长/周月年报告等）、**回忆坐标**、**一起听状态**。
- 数字专辑与新碟、语种风格馆、已购单曲/专辑、付费精选相关接口。
- **设置**只读或配置类、国家编码列表、**内部版本接口**（文档注明）、歌单播放量更新等偏运营/客户端行为类接口。

## 4. 限制与风险（范围外或需额外设计）

1. **鉴权与 Cookie**：大量「我的」类接口依赖登录态（Cookie 等），浏览器直连音源不适合暴露凭据，需 **BFF** 或同源代理策略。
2. **稳定性与变更**：第三方封装随官方接口变更可能出现字段或行为差异；能力范围文档**不承诺**每条接口即时可用。
3. **协作边界**：本仓库内 Agent 以《任务书》IN/OUT 与 **AGENTS.md §5.2** 为执行参照；是否使用某条能力与法律/平台义务的权衡由维护者自担。
4. **非功能需求**：限流、审计、多租户、业务数据库等 **不在** Alger 包职责内，由自建 `server/` 补充。
